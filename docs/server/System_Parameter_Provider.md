Provides parameters received from the <tt>sys_param</tt> stream.

### Class Name

### Configuration

This service is defined in <tt>etc/processor.yaml</tt>. Example:

{% yaml processor.yaml %}
realtime:
  services:
    - class: org.yamcs.parameter.SystemParametersProvider
{% endyaml %}
